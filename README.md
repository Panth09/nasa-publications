A dynamic, front-end dashboard to analyze and explore NASA's space biology research publications. This application connects to a real-time Google Firebase database to make a large, complex dataset of scientific papers easily searchable, filterable, and digestible for researchers, students, and enthusiasts.

## Core Features

* **‚ö° Real-time Data Integration:** Connects directly to Google Firebase (Firestore) to dynamically load and display all publication data.
* **Advanced Filtering & Search:** Features multi-parameter filtering (by relevance, publication year, and experiment duration) and a debounced, full-text search for instant results.
* **üìä Interactive Analytics Dashboard:** A dedicated modal provides visualizations for key metrics, including a chart of publication trends over time.
* **üîç In-Page Preview Modal:** Allows users to read a publication's full abstract and key findings without leaving the main page.
* **üé® Multi-Theme UI:** Includes four user-selectable themes (Dark, Light, High Contrast, and NASA Blue) with user preferences saved to local storage.
* **üì• Data Export:** Functionality to export the currently filtered dataset directly to a CSV file.
* **‚ú® Engaging UI/UX:** Built with modern, responsive CSS and JavaScript, including interactive 3D-tilt effects on the developer profile cards for an engaging user experience.

## Technologies Used

* **Frontend:** HTML5, CSS3 (Flexbox, Grid, Animations), Vanilla JavaScript (ES6+ Modules)
* **Backend-as-a-Service (BaaS):** Google Firebase (Firestore) for the real-time NoSQL database.
* **APIs:** Firebase Web SDK
